Introduction

Thousands of houses are sold everyday. There are some questions every buyer asks himself like: What is the actual price that this house deserves? Am I paying a fair price? In this paper, a machine learning model is proposed to predict a house price based on data related to the house (its size, the year it was built in, etc.). During the development and evaluation of the model, the code used for each step followed by its output. This will facilitate the reproducibility of the work. In this study, Python programming language with a number of Python packages will be used.

Goals of the Study¶

The main objectives of this study are as follows:

  • To apply data preprocessing and preparation techniques in order to obtain clean data
  • To build machine learning models able to predict house price based on house features
  • To analyze and compare models performance in order to choose the best model

Learning Objectives:¶

  • Data Preprocessing: Gain experience in handling and cleaning real-world data. This includes dealing with missing values, outliers, and transforming variables.

  • Feature Engineering: Learn how to select, create, and transform features to improve model performance. This involves encoding categorical variables, scaling numerical variables, and potentially creating new features.

  • Regression Modeling: Understand various regression algorithms and techniques, including linear regression,knn, decision trees. Learn how to train, evaluate, and tune regression models.

  • Model Evaluation: Explore different evaluation metrics for regression tasks, such as mean squared error (MSE) and root mean squared error (RMSE).

  • Feature Importance Analysis: Learn how to interpret feature importance scores provided by the model. Understand which features have the most impact on predictions.

Data Preparation

In this study, we will use a housing dataset presented by De Cock (2011). This dataset describes the sales of residential units in Ames, Iowa starting from 2006 until 2010. The dataset contains a large number of variables that are involved in determining a house price. We obtained a csv copy of the data from https://www.kaggle.com/datasets/marcopale/housing. The dataset contains 2930 records (rows) and 82 features (columns).

For Nearest Neighbors, we will use an implementation of the k-nearest neighbors (KNN) algorithm provided by Scikit-Learn package.

KNeighborsRegressor(n_neighbors=5, weights=uniform, algorithm=auto,
